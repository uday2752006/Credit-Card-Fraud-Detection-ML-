Detect fraudulent transactions using XGBoost and anomaly detection techniques.

- Evaluation with confusion matrix & ROC

## How to Run
1. Clone the repo
2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```
3. Launch dashboard:
   ```
   streamlit run app.py
   ```
